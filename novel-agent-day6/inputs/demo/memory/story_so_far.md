1. 临江三中的林砚因父亲去世，放学后在便利店打工帮母亲分担压力，雨夜遇到带天台水泥粉、头顶悬着72小时倒计时的女孩苏未。苏未预言的三件事（墙砖砸电动车、母亲加班来电、便利店停电）全部应验，告知林砚三天后临江会发生“时坠”，要求他别让人接近旧教学楼钟房、明晚前找到“停摆碎片”、别信学生会长顾潮生，还称林砚原本会在三天后死亡，林砚下班后跟苏未前往学校。
2. 两人翻进学校，苏未解释“停摆碎片”是能稳定局部时间剩余值的道具，途中遇到顾潮生。顾潮生指出林砚曾因“时间回响”死过一次，苏未承认上一次林砚救她后身亡；走廊出现异象，苏未让林砚在钟房开门时将停摆碎片按进钟芯裂缝，自己负责拖住顾潮生。
3. 钟房打开后露出“时间坠井”，顾潮生与苏未争论上一次的实验，顾潮生称当时是为了避免时坠在市中心爆发才推进实验；林砚冲去钟芯，看到上一次自己死亡的画面，耳边响起父亲的声音，将碎片按进裂缝后，碎片反向吸收他的时间，苏未扑过来时两人被白光吞没。
4. 林砚醒来后发现苏未将自己的时间分给了他，苏未头顶倒计时只剩12小时左右；顾潮生说明这次只是延缓时坠，需找到林砚父亲带走的完整核心另一半“钥匙”才能彻底解决，林砚想到家中的旧怀表；顾潮生离开后，林砚背苏未回家，苏未提及林砚意识中断时额头出现金色刻纹，林砚决心救下苏未。
5. 林砚背苏未走老巷回家，趁母亲加班不在家，找到父亲留下的锈铁盒，盒中旧怀表被时间锁锁住，蛮力无法开启。
6. 苏未提醒时间锁认人，她触碰怀表后，怀表浮现金色刻纹，同时市立钟塔钟声慢半拍，周边出现短暂时间停滞异象。
7. 铁盒暗格弹开，掉出父亲的纸条，上面写着“别信第二个站在你面前的人”，林砚对相关人物产生怀疑。
8. 苏未触发时间锁反噬，头顶倒计时加速流逝，房间出现白光，林砚额头发烫，体内“锚”产生剧烈共鸣。
9. 苏未晕倒在林砚怀中，怀表开始倒转，同时传来模糊声音称“钥匙已经醒了，门很快就要开了”。
