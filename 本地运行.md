1. 启动数据库
```powershell
cd D:\github_projects\PlotWeaver
docker compose up -d
```

2. 启动后端 API
```powershell
cd D:\github_projects\PlotWeaver\apps\api
$env:DATABASE_URL="postgresql+psycopg://postgres:postgres@127.0.0.1:5432/plotweaver"
$env:DEFAULT_TENANT_ID="00000000-0000-0000-0000-000000000001"
$env:APP_ENV="dev"
$env:APP_DEBUG="true"
$env:API_PREFIX="/api/v1"
D:\github_projects\PlotWeaver\.venv\Scripts\alembic.exe upgrade head
D:\github_projects\PlotWeaver\.venv\Scripts\python.exe -m uvicorn plotweaver_api.main:app --reload --host 127.0.0.1 --port 8000 --app-dir src
```

3. 配置前端环境变量
在 `D:\github_projects\PlotWeaver\apps\web\.env.local` 写入：
```env
NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:8000/api/v1
NEXT_PUBLIC_TENANT_ID=00000000-0000-0000-0000-000000000001
```

4. 启动前端
```powershell
cd D:\github_projects\PlotWeaver\apps\web
npm install
npm run dev
```

5. 验证
- 打开 `http://localhost:3000/app/projects`
- API 健康检查：`http://127.0.0.1:8000/api/v1/health/live`
