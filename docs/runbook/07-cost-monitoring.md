# 07 成本监控

## 预算维度
1. Vercel 使用量（构建分钟、带宽、函数执行）。
2. 容器平台使用量（实例规格、运行时长、出口流量）。
3. Supabase 使用量（计算、存储、连接数、出口流量）。

## 阈值规则
1. 预警：月度费用达到预算 80%。
2. 严重：月度费用超过预算 100%。
3. 异常峰值：24 小时费用超过滚动均值 2 倍。

## 处理动作
1. 达到预警阈值：复盘流量与实例规格。
2. 达到严重阈值：冻结非关键发布并立即优化。
3. 出现峰值阈值：排查异常流量与高消耗接口。

## 报告机制
1. 每周成本摘要。
2. 每月预算偏差分析。
3. 每次 Run 成本趋势（优化核心指标）。
